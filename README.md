## Funcionalidades

* **Catalogo Dinâmico:** Listagem de produtos consumindo API do backend.
* **Detalhes do Produto:** Pagina exclusiva com descrição, preçoo e foto.
* **Carrinho de Compras:** Gerenciamento de estado global via Context API.
* **Responsividade:** Layout adaptado para Desktop e Mobile.

## Como Rodar o Projeto

### Pre-requisitos
* Node.js instalado
* Git instalado

### 1. Clonar o repositorio

```bash
git clone https://github.com/raulscarvalho/rskbuys-project.git
cd rskbuys-project
```

### 2. Configurar o Backend (Server)

```bash
cd server
npm install
# Crie um arquivo .env na raiz da pasta server com:
# MONGODB_URI=sua_string_de_conexao_mongo
node src/app.js
```
*O servidor rodará em: `http://localhost:3000`*

### 3. Configurar o Frontend (Client)

Abra um novo terminal na raiz do projeto:

```bash
cd client
npm install
npm run dev
```
*O site abrirá em: `http://localhost:5173`*
